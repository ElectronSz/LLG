# LLG?

Get away with typing console.log() when logging data

# Installation

`npm i llg --save`

Example...

```
var lg = require('llg').llg

lg("your message")

```

Or 

```
import { llg } from 'llg'

lg("your message")

```

## Options

- dafault (param) is null
- pass arrays, objects and more
